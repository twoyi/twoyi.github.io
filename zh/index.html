两仪

一个轻量级的 Android 容器

开始了解

完全可定制

内部为完全可定制的 Android 8.1 系统，可支持 Root、Xposed 等。

免 ROOT

可以在免 ROOT 的手机上刷入 ROM，体验玩机的乐趣。

开源（正在进行中）

两仪 App 和内部系统将在 github 开源，可以从开源代码编译属于你自己的 ROM！
